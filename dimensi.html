Dimensi Cloud - Cloud Computing

Overview

Dimensi Cloud is a cloud computing solution with cloud infrastructure that can be customized according to needs. DimensiCloud supports business needs that require high traffic, with customizable storage capacity and the use of AI in cloud management so that it will produce more optimal performance.

Problem & Challenge

Dimensi Cloud need a web profile to present service information and advantages to increase engagement with clients. This website must present information clearly and in a modern style.

Solution

To design this website, I conducted benchmarking and discussed further with stakeholders regarding the details of the needs so that the results are achieved as expected.
  • Create a service customization simulation section
  • Provide technical assistance information
  • Ensure responsive design with desktop and mobile views

Result & Impact

This website was successfully launched and can help marketing and business activities from Dimensi Cloud.
  • Comfortable design and interaction
  • Information presentation is easy to navigate

Conclusion

This project improved my ability to communicate and collaborate across divisions. In addition to project managers and users, I also worked closely with marketing and graphic design teams to create representative assets.
